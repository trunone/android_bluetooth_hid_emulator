# Android Bluetooth HID Emulator

This project is an Android application that allows your device to act as a Bluetooth HID (Human Interface Device), such as a mouse or keyboard, to control other devices.

## Features

- Bluetooth HID Profile implementation.
- Mouse and Keyboard emulation.
- Device discovery and pairing.

## Getting Started

### Prerequisites

- Android device with Bluetooth HID support (Android 9.0 Pie / API level 28 or higher).
- Android Studio for building and deploying the application.

### Building

1. Clone the repository.
2. Open the project in Android Studio.
3. Build and run the `app` module on your Android device.

## Usage

1. Enable Bluetooth on your Android device.
2. Open the app and select a paired device or scan for new ones.
3. Use the interface to control the connected device via HID.
